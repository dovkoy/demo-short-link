# Chatform - Short Link feature

POC short link feature implemented with Next.js and Supabase Postgres.

## Quick Start

Please use the UI to sign-up and experiment with the link shortening feature.

## Deployment

- The Supabase project is [linked](https://supabase.com/docs/reference/cli/supabase-link) to a live, hosted Supabase project.
- Next.js frontend is deployed on vercel (deployment link to be shared or viewed on Github).

## Process

### Step 1
I started with:
- [This](https://github.com/nextjs/saas-starter) Next.js template, which I already fixed to successfully deploy on Vercel
- A blank Supabase project, with migration and seed files which I generated to match the Next.js template's schema

Please check the Next.js template [readme](nextjs/README.md) for details about that project and how to get it running locally, if so desired.
